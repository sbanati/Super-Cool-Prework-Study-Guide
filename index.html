HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is also defined, and sets the title of the webpage.
  • The user won't see information in the head or meta elements

Git

  • git status tells us what branch we are on
  • git checkout -b feature/"name" creates a new branch and moves us to it
  • git add . or git add -A or git add "file name" adds new code content to branch stage area
  • git commit -m "added new feature" adds a comment on the staged changes that can be viewed in project history
